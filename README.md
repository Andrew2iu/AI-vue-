# AI 心理助手前端项目

## 项目介绍

此系统是一个基于AI技术的心理健康助手前端应用，集成了先进的自然语言处理和情绪分析技术，实现从用户注册登录、情绪日志记录、智能AI咨询对话到知识文章管理的全流程功能。系统采用企业级开发规范，具备完整的用户权限控制和数据安全机制，为用户提供个性化的心理支持服务。

### 核心AI功能

- **智能对话系统**：基于流式API实现实时AI回复，提供流畅的对话体验
- **情绪分析引擎**：实时分析用户情绪状态，提供个性化的情绪管理建议
- **智能推荐系统**：根据用户情绪和咨询历史，推荐相关心理健康知识
- **会话管理**：智能管理用户咨询会话，支持多会话并行处理

### 技术亮点

- 采用最新的AI流式传输技术，实现毫秒级响应
- 集成情绪识别算法，提供精准的情绪分析
- 基于Vue 3 Composition API构建，代码结构清晰可维护
- 响应式设计，适配多种设备
- 完善的错误处理和用户体验优化

## 技术栈

### 前端技术

Vue 3 + Element Plus + Pinia + Vue Router + Axios + WangEditor + Vite + Day.js + ECharts + SCSS

### AI相关技术

流式API + 情绪分析 + 自然语言处理 + 数据可视化 + 会话管理

## 性能优化

### 前端性能优化

- **路由懒加载**：将路由组件改为懒加载形式，降低页面初始加载资源体积，提升首屏加载速度，尤其针对情绪分析图表等复杂组件
- **图片懒加载**：配置图片懒加载，自定义 loading/error 占位图并优化图片资源，减少非首屏图片的无效请求，提升用户体验
- **组件按需加载**：Element Plus 组件按需引入，减少打包体积
- **代码分割**：使用 Vite 的代码分割功能，实现模块的按需加载

### AI性能优化

- **流式API优化**：采用流式传输技术，实现AI回复的实时展示，减少用户等待时间
- **情绪分析计算优化**：使用防抖和节流技术，避免频繁的情绪分析计算
- **缓存策略**：对会话数据和情绪分析结果进行缓存，减少重复请求
- **计算密集型任务处理**：将复杂的情绪分析计算放在后端进行，前端只负责展示结果

## 资源优化

- **静态资源配置**：合理管理图片和字体资源，降低服务器带宽压力
- **SEO优化**：优化页面 meta 标签，补充 description、keywords 等 SEO 标签及正确语言属性，提升页面搜索引擎友好度
- **缓存策略**：合理使用缓存策略，减少重复请求，提升系统响应速度
- **静态资源压缩**：使用 Vite 的压缩功能，减小静态资源体积
- **字体优化**：使用系统字体或按需加载字体，减少字体文件大小

## 项目概述

这是一个基于 Vue 3 + Element Plus 开发的 AI 心理助手前端项目，集成了先进的人工智能技术，为用户提供个性化的心理健康服务。项目采用现代化的前端技术栈，实现了以下核心功能：

- **智能情绪分析**：基于AI算法实时分析用户情绪状态，生成情绪趋势报告，提供个性化的情绪管理建议
- **智能对话系统**：采用流式API技术，实现与AI助手的实时对话，提供专业的心理支持
- **知识推荐系统**：根据用户情绪和咨询历史，智能推荐相关心理健康知识
- **情绪日志管理**：记录每日情绪状态，分析情绪变化趋势，提供情绪统计报告
- **用户认证与权限**：安全的用户注册、登录系统，完善的权限管理机制
- **数据可视化**：使用ECharts实现情绪趋势、用户活跃度等数据的可视化展示
- **后台管理系统**：数据统计仪表盘、内容管理、用户管理等功能

## 功能模块

### 1. 用户认证

- **登录/注册功能**：安全的用户认证系统，支持密码加密存储
- **权限管理**：基于角色的权限控制，确保系统安全
- **身份验证**：使用JWT令牌进行身份验证，保障用户数据安全

### 2. 智能情绪分析

- **情绪日志记录**：用户可记录每日情绪状态，包括情绪类型、强度和相关事件
- **AI情绪分析**：基于AI算法实时分析用户情绪状态，提供情绪评分和风险评估
- **情绪趋势分析**：使用ECharts实现情绪变化趋势的可视化展示
- **个性化建议**：根据情绪分析结果，提供个性化的情绪管理建议

### 3. 智能对话系统

- **实时AI对话**：采用流式API技术，实现与AI助手的实时对话，提供专业的心理支持
- **会话管理**：智能管理用户咨询会话，支持多会话并行处理
- **情绪识别**：在对话过程中实时识别用户情绪变化，调整回复策略
- **对话历史**：保存对话历史，方便用户回顾和分析

### 4. 智能知识推荐

- **个性化推荐**：根据用户情绪和咨询历史，智能推荐相关心理健康知识
- **文章分类**：按主题和情绪类型对知识文章进行分类
- **文章详情**：提供详细的心理健康知识文章，支持富文本展示

### 5. 数据可视化

- **情绪趋势图表**：展示用户情绪变化趋势，帮助用户了解自己的情绪状态
- **用户活跃度分析**：分析用户使用频率和活跃度，提供使用建议
- **咨询会话统计**：统计咨询会话数量、时长和用户参与度

### 6. 后台管理系统

- **数据统计仪表盘**：实时展示系统运行数据，包括用户数、咨询量、情绪分布等
- **内容管理**：管理知识文章、情绪标签等内容
- **用户管理**：管理用户账号、权限和使用情况
- **系统监控**：监控系统运行状态，确保系统稳定

## 快速开始

### 环境要求

- Node.js >= 16
- npm >= 8

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录

### 预览生产版本

```bash
npm run preview
```

## 配置说明

### 环境变量

在 `.env` 文件中配置以下环境变量：

```env
# API 基础 URL
VITE_API_BASE_URL=http://localhost:3000

# 其他配置...
```

### 路由配置

路由配置文件位于 `src/router/index.js`，包含前端路由和后台管理路由。

### API 接口

API 接口封装在 `src/api` 目录下，分为前端接口和后台接口。

## 部署说明

1. 构建生产版本：`npm run build`
2. 将 `dist` 目录上传到服务器
3. 配置服务器指向 `dist` 目录
4. 确保服务器支持静态文件服务
